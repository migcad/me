
Headings

The levels are meant to form a logical hierarchy.
Markdown headings are fundamentally structural containers, not just visual styling because many systems automatically generate side navigation from headings.

```none
# H1 (Main title)
## H2 (Major sections)
### H3 (Subsections)
#### H4 (Occasional deep detail)
##### H5
###### H6
```

Markdown itself only defines H1 through H6.  There is no H7, no H8, etc.
