# Artificial Intelligence Ontology

## Introduction

An Artificial Intelligence system is better represented as a **multidimensional ontology** rather than a single hierarchical taxonomy.

A single hierarchy creates classification problems because AI concepts exist at different ontological levels.

For example:

* **Transformer** describes a computational architecture.
* **Text** describes a data modality.
* **Classification** describes a computational objective.
* **Supervised Learning** describes a learning paradigm.
* **Discriminative AI** describes the relationship between input data and output data.

These concepts should not compete for the same position in a single hierarchy.

A rigorous ontology separates AI systems into independent dimensions where each dimension answers one specific classification question.
